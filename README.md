# Minr Search Template

A reusable `search` namespace for [Minr](https://minr.org) mapmakers. It makes *search maps* (where the player has to find a bunch of hidden things) easy to set up without writing any complicated scripts — the library handles counting, per-player progress, messages, sounds, and teleporting on completion.

See the full guide, with a downloadable example, at [ak1089.github.io/minr/search](https://ak1089.github.io/minr/search/).
